# `DS1621` library

Arduino library for communication with a DS1621

## Usage
- Download and copy the `DS1621` folder to your Arduino libraries folder (usually `C:\Users\<user_name>\Documents\Arduino\libraries`).
- Include the library in your Arduino code:
```
#include "DS1621.h"
```
